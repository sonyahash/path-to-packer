Overview: This runbook will outline the steps required to configure an ubuntu instance on AWS with an NGINX application.

Creating Ubuntu Server in AWS

  1. Create a new instance by using the Amazon EC2
    1. In your AWS environment click “Instances” → “Launch Instances” (the orange button)
    2. Give your instance a name under the “Names and tags”
    3. Under the “Application and OS Images (Amazon Machine Image), select “Ubuntu Server 20.04 LTS (HVM), SSD Volume Type”
    4. Choose the instance type you want (i.e. t2.micro)
    5. Create a key pair
    6. Under Network Settings, find the Firewall section and click “create security group”
      1. Check all three boxes to create rules for SSH, HTTPs & HTTP traffic
    7. For the “Configure storage,” “Advanced details,” and “Summary” sections, use the default settings
    8. Click “Launch instance” at the bottom of the page
    9. Connect to the instance by clicking on the Instance ID → “Connect” → “Connect”
